PHP WebDriver
=============

`Documentation <doc/index.rst>`_

A library to manipulate a WebDriver server.

Installation
::::::::::::

Add the library to your **composer.json**:

.. code-block:: yaml

    {
        "require": {
            "alexandresalome/php-webdriver": "~0.3"
        }
    }
